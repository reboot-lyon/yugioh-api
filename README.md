# :+1: Typescript template API
NODE API template in TypeScript.
## :rocket: Installation
>**Manuel installation**
```
npm install

npm run dev
```
## :round_pushpin: EndPoints
### **[GET]** /api
_Foo Bar services_

```
}
```

>Output structure example
```
{
    res: {
        foo: 'bar'
    }
}
```
